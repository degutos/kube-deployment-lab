# kube-deployment-lab
Lab with kubernetes using mongodb and mongo-express with internal and external services using minikube - From Nana tecnoworld


# Overview

This is a simple lab I've used on my studies with kubernetes. This lab is from Nana of tecnoworld channel 
Youtube: https://www.youtube.com/watch?v=X48VuDVv0do

This lab will deploy a pod using MongoDb image and Mongo-Express application accessing our DB Mongo. We will also use external and internal service also configmag storing variables and secret to manage credentials.


# Components 

- 2 Deployment / Pod
- 2 Services
- 1 ConfigMap
- 1 Secret


## Detailing

- MongoDB Pod 
- Internal Service for MongoDB
- ConfigMap with DB URL
- Secret with DB User and DB PWD
- Mongo-Express Pod
- External Service exposing Mongo-Express
